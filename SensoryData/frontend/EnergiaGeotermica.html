Energía Geotérmica

El grupo 7 presenta una simulación de una planta de energía geotérmica que utiliza un circuito electrónico con un motor a pasos y un ventilador para replicar la conversión de energía térmica en electricidad. Un ESP32, configurado como estación y servidor web, recopila datos de voltaje, corriente y potencia, almacenándolos en una base de datos MySQL y permitiendo su consulta a través de una API REST. La maqueta está integrada con Unity, donde los datos se visualizan en tiempo real mediante medidores y representaciones interactivas, proporcionando una experiencia educativa que facilita la comprensión del proceso de generación de energía geotérmica.
